# Spawn
## The Spawn Engine is a stateless virtual DOM generator.

## Use
1. npm install
2. Import module in your app e.g.

```js
import Spawn from '@nurvus/spawn';

const mountEl = document.body;
Spawn({
  parentEl: mountEl,
  children: [
    'Your Spawn is ready.'
  ]
});
```

## Contribute
1. `npm install` Install
2. `npm run demo` Test
